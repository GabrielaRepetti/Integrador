El elemento (iframe) en HTML se utiliza para incrustar otro documento HTML dentro del documento actual. Es como colocar una ventana dentro de tu página web que muestra otra página o contenido externo.

Uso común:

Incrustar videos:Muy usado para insertar videos de YouTube, Vimeo, etc.
Insertar mapas:Por ejemplo, un mapa de Google Maps mostrando la ubicación de una tienda.
Cargar formularios externos:Como un formulario de Google Forms o Typeform.
Publicidad:Muchos anuncios se cargan en (iframe) para aislar su contenido y evitar conflictos con el sitio principal.
Mostrar documentos PDF u otro contenido.
