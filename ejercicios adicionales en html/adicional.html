Etiquetas semánticas:

Nuevo conjunto de etiquetas(header,nav,section,article,aside,footer,main,figure,figcaption,mark,time,data,output,progress y meter).Estas aportan significado al contenido,mejoran la accesibilidad.
Multimedia nativa: HTML5 incorpora soporte directo para audio y video con las etiquetas (audio y video), eliminando la necesidad de plugins externos como Flash.
Canvas y gráficos vectoriales: La nueva etiqueta (canvas) permite dibujar contenido 2D dinámico mediante JavaScript. También se integran SVG y soporte 3D como WebGL en algunos contextos.
Mejoras en formularios: Nuevos tipos de (input como email, url, tel, date, datetime-local, month, week, time, number, range, color, search), con validación incorporada. Además, atributos como (placeholder, required, pattern, min, max, y elementos como datalist) para autocompletar.
API de arrastrar y soltar (Drag & Drop): Permite crear interfaces interactivas usando arrastrar elementos dentro de una página sin JavaScript externo adicional.
Almacenamiento local y trabajo fuera de línea: Con localStorage y sessionStorage, las aplicaciones web pueden almacenar datos localmente en el navegador. También se incluyen APIs para trabajar offline, como AppCache y Service Workers, que permiten usar aplicaciones sin conexión.
Web Workers y procesamiento en segundo plano: Permiten ejecutar scripts en hilos separados sin bloquear la interfaz principal, mejorando rendimiento y experiencia de usuario.
WebSockets y Server-Sent Events: Tecnologías para comunicación en tiempo real.
WebSockets: conexión bidireccional persistente entre cliente y servidor.
Server-Sent Events (SSE): flujos unidireccionales desde el servidor al cliente.
API de Geolocalización: Permite obtener la ubicación del usuario, útil para aplicaciones que ofrecen servicios basados en posición geográfica como mapas, recomendaciones o comercio local.
Elementos y atributos globales: Nuevos atributos variables como data, hidden, async, tabindex, charset en meta, y la eliminación de muchos elementos y atributos obsoletos como (font,center,big,basefont,applet,frame,frameset,strike), entre otros.
Otras APIs integradas: También se incorporaron APIs como History (para manipular la barra de navegación), registro de controladores MIME, Microdata (para metadatos estructurados), Messaging, y formatos embebidos como SVG y MathML
HTML: Aunque HTML evoluciona más lentamente, hay algunas novedades en discusión y en adopción temprana: Declarative Shadow DOM: Ya soportado en algunos navegadores. Permite usar Shadow DOM sin JavaScript. Popovers: Nuevo elemento nativo para crear popups tipo tooltips, menús, etc. Mejor soporte cruzado en 2025. Web Components mejor integrados: Se promueve el uso de (template y slot) con mejoras en compatibilidad y performance.
CSS: CSS ha avanzado muchísimo. Estas son algunas de las características más destacadas que están siendo ampliamente adoptadas:has() selector (CSS “parent selector”):Permite seleccionar un elemento padre si contiene un hijo específico.
Container Queries: Ya están ampliamente soportadas. Permiten aplicar estilos basados en el tamaño del contenedor, no del viewport.
Style queries: Consulta condicional de estilos aplicados. Aún en desarrollo.
CSS Nesting:Permite anidar selectores al estilo de Sass.
New color spaces: Compatibilidad creciente con color(display-p3), lab(), oklab() para colores más precisos y vibrantes. Scroll-driven animations (@scroll-timeline):Nativas en CSS sin JS. Se pueden crear animaciones sincronizadas con el scroll. JavaScript (JS/ECMAScript)La evolución de JavaScript también continúa con propuestas y mejoras clave:
Temporal API (fecha y hora mejorada): Sustituto moderno de Date. Mucho más potente y preciso. const now = Temporal.Now.zonedDateTimeISO();Records & Tuples (propuesta avanzada): Tipos inmutables similares a estructuras o arrays/objetos puros.Pattern Matching (propuesta Stage 3): Permitiría destructurar y evaluar estructuras con más facilidad, estilo switch.Top-level await:Ya ampliamente soportado en módulos ES. Permite usar await sin funciones async.
WebAssembly y WebGPU: Cada vez más integrados al ecosistema JS para rendimiento gráfico y computacional avanzado. Decorators (Stage 3 - Estables en algunos frameworks):Permiten modificar clases y propiedades de forma declarativa(Angular, Lit,etc.).
